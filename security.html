Security, privacy, and permissions for Grab All Files

Grab All Files needs broad browser permissions because it can scan the page or URL you choose, including pages behind your current login session. The extension is designed so file discovery, downloads, ZIP creation, CSV export, and PDF merging happen locally in your browser.

Grab All Files は、ユーザーが指定したページやURLをスキャンするため、ブラウザ上で広い権限表示が出ることがあります。ただし、ファイル検出・ダウンロード・ZIP作成・CSV出力・PDF結合はブラウザ内で実行され、ファイル本文やページHTMLを当方サーバーへ送信しません。

Plain-language summary / 要点

  • 1Your files never pass through our servers.
  • 2PDF merge, ZIP, and CSV export run locally in the browser.
  • 3License checks send only license-related identifiers, not file contents.
  • 4The extension should be used only for pages and files you are allowed to access.

What is sent to our servers

Only license and trial verification data is sent to the developer's license backend.

PurposeData sentNot sent
Trial eligibilityRandom device UUID and timing needed to check the 14-day trial.File contents, page HTML, browsing history, cookies, passwords.
License activation and refreshLicense key, random device UUID, and license token metadata.Downloaded files, detected URLs list, source-page contents.
Revocation list refreshNo user-identifying payload; the extension downloads a signed revocation list.License key and file data are not included in this request.

当方サーバーへ送信されるもの

送信されるのは、無料トライアルとライセンス確認に必要な情報だけです。ファイル本文、ページHTML、閲覧履歴、Cookie、パスワード、検出したURL一覧は送信しません。

Why broad website permissions are needed

Browsers may show a warning that the extension can access website data. Grab All Files needs this because the user may choose any target site, and the extension must inspect the selected page, frames, file responses, and download links before it can list files.

Permission areaWhy it is neededHow it is limited in practice
Read the selected pageFind visible links, embedded files, PDF viewers, and lazy-loaded assets.Used for pages or URLs the user chooses to scan.
Monitor file-like network responsesSurface files loaded through fetch/XHR, CMS routes, or viewer endpoints.Used to build the local result list; the list is not uploaded to our server.
Use the current browser sessionDownload files from internal portals, LMS pages, or member areas that the user can already open manually.No separate credential collection; use only where you have permission to access the content.
The extension does not grant access to content you are not authorized to view. It works with access your browser session already has.

広いWebサイト権限が必要な理由

ブラウザ拡張の権限表示では強く見えることがありますが、対象サイトは事前に固定できないため、ユーザーが選んだページ・iframe・ファイルレスポンス・ダウンロードリンクを調べる権限が必要です。これは、権限のないコンテンツへアクセスするためのものではありません。

  • PDF merge processing.
  • ZIP archive generation.
  • CSV export generation.

ローカルに残る処理

  • ダウンロードしたファイル本文。
  • PDF結合処理。
  • ZIP作成処理。
  • CSV出力処理。

Responsible use

Grab All Files is intended for legitimate document collection and workflow automation: research, public records, internal portals, LMS materials, design assets, and similar work. Use it only for files you are allowed to access and download, and respect the target site's terms, copyright rules, and rate limits.

責任ある利用

Grab All Files は、研究資料、公開資料、社内ポータル、LMS教材、デザイン素材など、正当にアクセスできる資料を効率よく収集するためのツールです。対象サイトの利用規約、著作権、アクセス権限、レート制限を守って利用してください。

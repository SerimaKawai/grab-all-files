🔎

Finds files others miss

708 file types (extensions) supported. Detects iframe-embedded PDFs, lazy-loaded images, fetch/XHR file responses, blob URLs, CMS download routes, hidden form values, and 39 data-* attributes that plain link scrapers ignore. Also reaches into Shadow DOM / Web Components, SPA hydration data (Next.js, JSON-LD), Cache Storage / OPFS / IndexedDB, and signed cloud-storage URLs (Cloudflare R2, Firebase, Supabase, Vercel Blob).

⚙️

Smart filtering

Filter results by extension, file size, file name, page title, or modification date. Sort, manually reorder, and copy URLs as plain text.

🕸️

Same-site crawl

Choose a depth (0–5 or unlimited) and visit every same-domain link to surface PDFs on subpages, viewer URLs, and CMS document routes — up to 100,000 pages per run. It also reads sitemap.xml / robots.txt and follows JavaScript-menu (javascript:/onclick), frameset, image-map, and GET search/pagination links, reaching pages that have no plain HTML links.

📚

Local PDF merge

Combine selected PDFs into a single file with one click. No upload, no server. Your files stay on your device.

🗜️

ZIP archive

Bundle selected files into a single ZIP for tidy distribution, archiving, or sharing — generated entirely in your browser.

📊

CSV export

Export URLs, file names, sizes, types, and titles to CSV for documentation, audit, inventory, or downstream automation.

🔐

Scans pages you can access

Uses your current browser session to scan intranets, LMS portals, and member-only pages you are already authorized to open. It does not collect credentials or override access controls.

🛡️
